# "Easy Clicker": Right-Click Auto Clicker

I built this mod because I couldn't find a simple auto clicker that did what i wanted it to do.

## What it does

This is a lightweight, vanilla-friendly auto clicker exclusively for right-clicking.

Features:
* **Two Modes:**
    * **Holding:** Clicks like you are holding down the right mouse button.
    * **Individual:** Variable Clicks per Second (CPS)
* **In-Game Config:** You can adjust your Target CPS and swap modes via a custom settings screen.
* **Persistent Settings:** Settings are saved to a config file.

## Keybinds

By default, the mod uses the following keys (these can be changed in the standard Minecraft controls menu):

* **`I`** - Toggle the Auto Clicker On/Off
* **`O`** - Open the Configuration Screen

## Technical Info

This is a Fabric mod built for 1.21.1. No other mods needed. Works in multiplayer too.
